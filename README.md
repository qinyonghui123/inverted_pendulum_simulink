# inverted_pendulum_simulink
Inverted pendulum swingup simulink

Most of the inverted pendulum swingup use Bang_Bang control.While,this swingup logic module is a position-limit control module. The input variables of this module are: slider position x/m; slider speed v/(m/s); pendulum angle Î¸/rad; pendulum angular velocity w/(rad/s).Stability of the pendulum is archived by LQR control.Provide both continuous and discrete systems.
